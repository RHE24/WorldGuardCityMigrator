Commands: [required] <optional>
/wgmigrate
Displays a list of player commands
/wgmigrate info <region>
List everything WGCM knows about the given region (or if none, your current WGCM-selected region) in the context of it being a city you might want to move. This includes the number of regions that would be moved with it, and its approximate size.
/wgmigrate region [region]
Selects a region in the current world for migration.
/wgmigrate from <x y z>
Set the "from" reference point used for calculating the delta. If no coords given, this uses your current location.
/wgmigrate to <x y z>
Set the destination reference point used for calculating the delta. If no coords given, this uses your current location. Make sure you use the same physical reference point in your city that is being moved for setting both the "from" and "to" points so that the regions and blocks line up exactly.
/wgmigrate delta [x y z]
You can use this command to directly input how much to move the city instead of using the "from" and "to" commands. For example, using "/wgmigrate delta 0 0 0" would result in no movement.
/wgmigrate rotation [degrees]
Set the clockwise rotation to be applied when migrating. Use increments of 90 degrees. Negative values will work. Will rotate around the reference point specified in "from". If you specify only a delta and no From, the city will be rotated around a calculated center point (not recommended).
/wgmigrate status
Show everything you have input for the current migration, indicating if it is enough to do the migration yet. Also provides some hints about the proposed migration such as any regions containing your target region in both the source and destination positions, as well as everything in the info command.
/wgmigrate process
Do the migration.
/wgmigrate undo
Undo your last migration. Only the last migration you did is remembered for undoing so do be careful. And you can't undo the undo.
/wgmigrate cancel
Clear your current selections in case you input some info and decided not to do it. That way you don't accidentally move things if you type "/wgmigrate process" later without thinking.
/wgmigrate reload
Reload the plugin.
Requires a separete reload permission
/wgmigrate version
Show curent version.
